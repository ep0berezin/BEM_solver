Program to solve Poisson equation boundary value problem by solving Fredholm equation of the 2nd kind with collocation method.

Features:

- Reading meshes in gmsh2 format and INM RAS custom format
- Solving equation with user defined Neumann/Dirichlet BC and integral operator kernel
- Linear system is being solved with user-defined solver
- Visualization in PyVista and Matplotlib
- Contains an example with BVP solution for Darcy's law equations around a sphere (See Sphere_example.py)

Much can be improved. This version was finished in Autumn of 2025.
